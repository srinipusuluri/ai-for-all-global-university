LangFlow — Build & Ship AI Agents and RAG Apps, Visually

LangFlow is a low‑code, open‑source framework to design AI workflows, agents, and MCP servers with a drag‑and‑drop editor. It supports major LLMs, vector databases, and exposes flows as APIs.

Quick Install
# Using pip
pip install langflow

# Or with uv (recommended in docs)
uv pip install langflow -U

# Run locally
langflow run  # or: uv run langflow run

What is LangFlow?

It’s a visual graph editor where each node is an executable component (prompting, tools, retrievers, memory, agents, etc.). Connect nodes, configure params, run, and deploy as an API or MCP tool.

Learn LangFlow (Self‑Study)

A curated learning path you can follow at your own pace.

  1. Install and launch locally (see Quick Install).
  2. Complete the Quickstart (load a template flow, run, then call /run API).
  3. Build a basic RAG pipeline (ingest docs → retriever → LLM → output).
  4. Add tools & memory; experiment with agentic patterns and MCP.
  5. Deploy: export and integrate the flow via REST or as an MCP tool.

Sample Flow Pattern

Try this minimal RAG‑style outline to get comfortable with core blocks:

# 1) Loader/Indexer → 2) Vector Store (FAISS, Pinecone, Astra DB, etc.)
# 3) Retriever → 4) LLM (OpenAI, Groq, etc.) → 5) Output
# Tune: chunk size, top‑k, temperature. Add memory/tools as needed.

# Hitting a deployed flow via REST (after exporting/serving)

  "input": {"question": "Summarize policy.pdf"},
  "config": {"temperature": 0.2}
