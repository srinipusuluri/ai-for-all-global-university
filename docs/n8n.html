n8n — Automate Anything with Visual Workflows

n8n is an extensible, open‑source workflow automation tool that connects apps, APIs, and your code. Build complex workflows with a drag‑and‑drop UI and run them on your infra.

Quick Start
# Docker quickstart
docker run -it --rm \
  -p 5678:5678 n8nio/n8n

# Node.js (local)
npm install -g n8n
n8n start

# Visit the UI
http://localhost:5678

What is n8n?

Low‑code automation with 400+ integrations. Orchestrate cron jobs, webhooks, ETL, and AI agents that call tools/APIs.

n8n for Agentic AI

Use n8n to wrap AI models, retrieval steps, and external tools as nodes. Trigger agents via webhooks, pass context, and log outputs. Great for glueing SaaS + AI workflows.

  • Tools: HTTP Request, Code, Database, Files, custom.
  • RAG: call vector DBs and LLMs; route on confidence.
  • Deploy: cron, event, or API triggers; observability built‑in.

Learn n8n

  1. Run locally (Docker or Node).
  2. Build a webhook → transform → Slack/Email workflow.
  3. Add an LLM step for drafting responses.
  4. Persist state in a DB; add error handlers & retries.
