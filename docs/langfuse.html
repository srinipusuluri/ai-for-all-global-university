Langfuse — Observability for LLM & Agentic Apps

Langfuse is an open‑source telemetry platform for LLM applications. Capture traces, spans, prompts, costs, latencies, user feedback, and evaluations to improve quality and ship faster.

What Langfuse Tracks

  • Traces & Spans: capture request trees across tools/agents.
  • Prompts & Outputs: store templates, variables, and generations.
  • Tokens/Cost/Latency: monitor usage per model/user/feature.
  • Feedback & Scores: thumbs, rubrics, model‑based evals.
  • Datasets & Experiments: A/B prompts, model choices, parameters.

Fit in Agentic AI

Langfuse provides the observability backbone for multi‑step agents. Instrument each tool call and decision node to see why the agent behaved a certain way and how to improve it.

  • Trace chains/graphs (LangChain/LangGraph), orchestration (LangFlow/n8n), and custom agents.
  • Attach feedback loops to train evaluators or reward models.
  • Correlate failures with context (RAG top‑k, tool errors, timeouts).

Instrumentation Patterns

  1. Wrap each LLM/tool call with a span; include inputs/outputs & metadata.
  2. Group spans into a trace per user action or job.
  3. Send user feedback and evaluator scores back to Langfuse.
  4. Slice dashboards by model, feature, and prompt version to find regressions.

Example (Python sketch)

"""
Illustrative snippet — check official docs/SDK for latest.
"""
from time import perf_counter

class LFSpan:
    def __init__(self, name):
        self.name=name; self.t0=perf_counter()
    def __enter__(self): return self
    def __exit__(self, *exc):
        dt = perf_counter()-self.t0
        print({"span": self.name, "ms": int(dt*1000)})

with LFSpan("retrieve"):
    # call retriever/vector DB
    pass
with LFSpan("llm_call"):
    # call LLM
    pass

Governance & Privacy

  • PII handling & redaction before export to dashboards.
  • Access controls per environment (dev/stage/prod).
  • Token & cost budgets; alerts on anomalies.
  • Retention settings for traces/artifacts.
